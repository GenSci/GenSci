Coming Soon! Gentlemen Scientists.

Gentlemen Scientists aims to be a place of scientific collaboration and dissemination with a focus on marine science. We seek to share tools and data as well as methods and results.

Purpose

The purpose of Gentlemen Scientists is to disseminate scientific discoveries and ongoing research in marine and coastal sciences to the wider community. Additionally it seeks to facilitate collaboration and communication between scientists utilizing some of the tools and methodologies from the world of software development.

Goals

The goals of Gentlmen Scientsts are many and growing as needs or new use cases are identified. Some identified goals are as follows:

  • A blog in which scientific discoveries can be documented and research can be shared.
  • A set of code repositories to share any custom solutions developed to aid in research endeavors.
  • An application interface that allows registered scientists to store data in a geographic context and view publicly available data that may interact with their doman.
  • A community discussion forum where scientists and the greater public can actively communicate about past, ongoing, and future research.

Methods

Gentlemen Scientists is intended as a marriage between web application development and oceanographic study of remote sensing data. The site is developed using a Python 3 framework (Django) and thus much of the initial code base will likely be using that language.

Where possible, graphics will be interactive and directly interpret actual data, which will be available for download.

Participate

All those with sincere interest in this endeavor are welcome to participate.
